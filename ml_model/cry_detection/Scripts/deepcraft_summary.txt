The DEEPCRAFT
 ™ 
Ready Model for Baby Cry Detection detects babies and young 
children crying in the vicinity of a microphone in a wide range of indoor circum-
stances. The sound detection can then trigger special functionalities for various 
use cases.  
 
This model is targeted towards consumer electronics manufacturers that  
want to 
distinguish their products by adding intelligent functionalities. Using machine 
learning (ML) models for these features future
 -
proofs them, which is vital in a  
fiercely competitive consumer market. The model is designed to operate at opti-
mal efficiency in common environments, guaranteeing a  
smooth and effective 
user experience across a range of standard scenarios.  
 
While building machine learning models for the edge can be time consuming,  
the 
DEEPCRAFT
 ™ 
Ready Model for Baby Cry Detection lets companies avoid  
that com-
plex development. It is a fully trained, comprehensively tested Edge AI model that 
is ready for production. This leaves Infineon’s customers free  
to focus on how to 
best put to the model to use, and to commercialize the  
added value it brings to 
their product.
